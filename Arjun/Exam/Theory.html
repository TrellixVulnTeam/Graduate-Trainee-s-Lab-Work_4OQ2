Callback Function : A callback is passed as an argument to another function that executes the callback based on the result. They are basically functions that are executed only after a result.

Callback Hell : A Callback hell is essentially nested callbacks stacked below one another forming a pyramid structure. Every callback depends on the previous callback.

Question 5



In Pass by value, function is called by directly passing the value of the variable as an argument. So any changes made inside the function does not affect the original value.


In Pass by Reference, Function is called by directly passing the reference/address of the variable as an argument. So changing the value inside the function also change the original value

Question 6


Flexbox: Flexbox is a 1 dimensional system, that means, it can handle only rows. Example Header The main motive is to change width and height to fill the available space and its more focus is on content than layout it is used for small scale.

CSS Grid: CSS Grid is a 2 dimensional system, meaning it can handle both columns and rows Example : Body, Footer. It allows us to align components into columns and rows. It is mainly focused on layout other than content, grid is used for large scale where there is no linear display

All javaScript objects inherit properties and methods from a prototype. Prototype Inheritance, Date objects inherit from Date.prototype Array objects inherit from Array.prototype Person objects inherit from Person.prototype The Object.prototype is on the top of the prototype inheritance chain: Date objects, Array objects, and Person objects inherit from Object.prototype.
