Explain about destructing of an object in javascript. Destructuring Assignment is a JavaScript expression that allows to unpack values from arrays, or properties from objects, into distinct variablesdata can be extracted from arrays, objects, nested objects and assigning to variables. Example: var names = ["Samyuktha", "sahithi", "Karthik", "Bavish"]; var firstName = names[0]; var secondName = names[1]; console.log(firstName); console.log(secondName);
